# Rio

A polished React + Vite landing page for Rio, with a cinematic image carousel and a clean structure ready for deployment.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run lint`

## GitHub

1. Initialize Git:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   ```
2. Create a repository on GitHub and push:
   ```bash
   git branch -M main
   git remote add origin <your-github-repo-url>
   git push -u origin main
   ```

## Vercel deployment

This project includes a Vercel config file for SPA routing.

1. Install Vercel CLI if needed:
   ```bash
   npm i -g vercel
   ```
2. Deploy:
   ```bash
   vercel
   ```

For production deployment:
```bash
vercel --prod
```
